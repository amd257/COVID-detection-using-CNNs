Welcome to the COVID-19 X-ray Diagnosis Project



This project is developed by a student at Brunel University London to create a Web Application that uses Deep Learning models for diagnosing COVID-19 in patients. Different Deep Learning models were trained and evaluated on a dataset containing chest X-rays of COVID-19, Non-COVID Viral Pneumonia, other Non-COVID Lung Infections such as Bacterial Pneumonia, and Normal patients. The best performing Deep Learning models have been incorporated into this web application to make predictions.

Two types of classification approaches can be used in this application: COVID vs Non-COVID Detection and COVID Multi-Class Detection. For COVID vs Non-COVID Detection, all the Non-COVID chest X-rays are grouped together, so the model detects whether the chest X-ray uploaded is COVID-19 positive or negative. The most optimal Deep Learning model used for this approach is Xception, which has a 99.26% accuracy on average on unseen data. For the COVID Multi-Class Detection approach, all the different categories of chest X-rays are individually classified. So, the model will detect whether the X-ray uploaded shows signs of COVID-19, Non-COVID Viral Pneumonia, other Non-COVID Lung Infections or is the X-ray of a Normal patient. The Xception model is the best performing model for this classification approach as well, having a 93.15% accuracy on average on unseen data.
